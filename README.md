# Responsive Landing Page

A responsive landing page built using HTML, CSS, and JavaScript.

## Features

* Hero Banner Slider

  * Navigation arrows
  * Pagination dots
  * Autoplay
  * Infinite looping
  * Mobile swipe support

* Product Grid

  * 4 cards on Desktop
  * 2 cards on Tablet
  * 2 cards on Mobile

* Tabs Section

  * Active tab state
  * One tab visible at a time

* Accessibility

  * Semantic HTML
  * Keyboard navigation
  * Image alt attributes

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```text
├── index.html
├── style.css
├── script.js
└── README.md
```
